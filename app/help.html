File Stapler

v 1.0.0

1. Drop multiple files into the stapler and click ‘Staple’. These files are now stapled together.

2. To check which files are stapled together, drop a single file into the stapler and a list showing the different groups of stapled files will pop up. You can open any of the stapled files from this list.

3. For more options, open the main File Stapler interface. Here you can add files to groups, rename stapled groups, zip stapled files, and also unstaple groups.
